# 📖 Overview

This repository contains the principal figures and supplementary visual datasets associated with the DoloSil-20 study on **Stratospheric Aerosol Injection (SAI)** and the atmospheric engineering “SAI trilemma”:

- High optical efficiency  
- Thermal neutrality  
- Ozone safety  

The repository is structured to support manuscript preparation, peer review, archival transparency, and scientific reproducibility of figure assets.
