📖 Overview

This repository contains the principal figures and supporting visual PDFs associated with the DoloSil-20 study on Stratospheric Aerosol Injection (SAI) and the atmospheric engineering “SAI trilemma”:

High optical efficiency
Thermal neutrality
Ozone safety

The repository is organized to support manuscript preparation, peer review, figure archiving, and scientific transparency.
